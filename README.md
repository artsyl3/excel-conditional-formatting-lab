# excel-conditional-formatting-lab
A collection of Excel conditional formatting techniques used to analyze data, detect patterns, and improve visual reporting.

📊 Overview

This repository demonstrates practical use of Conditional Formatting in Microsoft Excel to analyze, highlight, and visualize data effectively.

The goal is to showcase how formatting rules can be used to quickly identify patterns, trends, and outliers in datasets.

🎯 Objectives

Apply conditional formatting to real-world datasets

Highlight key insights using visual rules

Improve data readability and decision-making

Build a structured portfolio of Excel skills

🛠️ Techniques Covered

Highlight Cells Rules (greater than, less than, duplicates, etc.)

Data Bars

Color Scales

Icon Sets

Custom formulas for dynamic formatting

Conditional formatting with dates

Conditional formatting based on other cells

💡 Example Use Cases

Highlight overdue dates

Identify top/bottom performers

Detect duplicates in datasets

Visualize trends using color scales

Flag important thresholds automatically

🚀 Why This Matters

Conditional formatting is a powerful tool for:

Faster data analysis

Cleaner reporting

Better visual communication in Excel

📌 Tools Used

Microsoft Excel (365)
